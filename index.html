🔑 Moonshot API 설정

    

      
      
      
      
      
    

    
키는 이 기기의 브라우저에만 저장됩니다. CORS 오류가 나면 아래 Cloudflare Workers 프록시를 만들어 URL을 입력하세요.

    

      🛠️ CORS 프록시 만들기 (2분)


      1. cloudflare.com 가입 (무료)

      2. Workers & Pages → Create Application → Create Worker

      3. 코드를 아래 내용으로 교체 → Deploy

      4. 생성된 URL을 위 프록시 URL에 입력


      Cloudflare Workers 코드:

      
export default { async fetch(request) { if (request.method === 'OPTIONS') { return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Methods': 'POST', 'Access-Control-Allow-Headers': 'Content-Type, Authorization' } }); } const response = await fetch('https://api.moonshot.cn/v1/chat/completions', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': request.headers.get('Authorization') }, body: request.body }); const r = new Response(response.body, response); r.headers.set('Access-Control-Allow-Origin', '*'); return r; } };
